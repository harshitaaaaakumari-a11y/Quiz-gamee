The Quiz Game is a console-based application developed in C language that allows users to test their knowledge through multiple-choice questions.
The project is built using Data Structures and Algorithms (DSA) to ensure efficient question management, score tracking, and performance optimization.
This project demonstrates the practical implementation of core DSA concepts in a real-world application.
Objective:
To develop a dynamic and efficient quiz game.
To implement advanced data structures like Linked List, Binary Search Tree, and Queue.
To optimize question storage, retrieval, and scoring using DSA techniques.
To provide an interactive and engaging learning experience.
